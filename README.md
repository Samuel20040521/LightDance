# LightDanceIDE
let library team have C++ IDE in neovim to use

# INSTALL
First  go to `~/.config`
```
mkdir nvim
cd nvim
touch init.lua
------- vim -------
require("LightDance.core")
require("LightDance.lazy")
------- vim -------
mkdir lua
cd lua
git clone https://github.com/Samuel20040521/LightDance
```
done!
